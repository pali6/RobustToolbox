Class UIController

Each UIController is instantiated as a singleton by Robust.Client.UserInterface.UserInterfaceManager UIController can use DependencyAttribute for regular IoC dependencies and UISystemDependencyAttribute to depend on EntitySystems, which will be automatically injected once they are created.
